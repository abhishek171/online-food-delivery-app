# Online Food Delivery Application Using REST API.

This project is made using spring boot(v 3.1.2)

Technology Stack

-> Core Java

-> Spring Boot Using JPA

-> MicroServices

-> MySql

Modules Used

-> Admin Module

-> Hotel Module

-> Customer Module

-> Delivery Module

-> Payment Module

FrontEnd

-> React Js

-> HTML

-> CSS

-> TailWind Css

Stripe -> Payment Gateway

FIrebase -> OTP Mobile Service
